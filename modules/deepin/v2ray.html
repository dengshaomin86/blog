安装 v2ray ​

1. 下载并安装 V2Ray ​

bash
wget https://raw.githubusercontent.com/v2fly/fhs-install-v2ray/master/install-release.sh
sudo bash install-release.sh

2. 配置 V2Ray: 编辑配置文件 /usr/local/etc/v2ray/config.json 来设置你的 V2Ray 服务器参数。你可以根据需要修改配置文件 ​

bash
# 打开配置文件
sudo nano /usr/local/etc/v2ray/config.json

# 编辑配置文件: 配置文件通常包括以下几个部分：
# 传入和传出设置：
{
  "inbounds": [
    {
      "port": 1080,
      "protocol": "socks",
      "settings": {
        "auth": "noauth",
        "udp": true
      }
    }
  ],
  "outbounds": [
    {
      "protocol": "vmess",
      "settings": {
        "vnext": [
          {
            "address": "your.server.address",
            "port": 10086,
            "users": [
              {
                "id": "your-uuid",
                "alterId": 64
              }
            ]
          }
        ]
      }
    }
  ]
}

# 更新 address、port 和 id：根据你的服务器信息修改
# 保存并退出: 按 Ctrl+X，然后按 Y 保存并退出编辑器

3. 重启 V2Ray 服务 ​

bash
sudo systemctl restart v2ray

4. 检查状态 ​

bash
sudo systemctl status v2ray

5. 启动 V2Ray ​

bash
systemctl start v2ray

6. 设置开机自启动 V2Ray ​

bash
systemctl enable v2ray

7. 停止 ​

bash
systemctl stop v2ray

8. 升级更新 ​

  • 在 VPS，重新执行一遍安装脚本就可以更新了，在更新过程中会自动重启 V2Ray，配置文件保持不变
bash
bash install-release.sh
