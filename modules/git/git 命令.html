git 命令 ​

1. 新建分支 ​

bash
git checkout -b xxx
# 等价于
git branch xxx
git checkout xxx

2. 推送新分支 ​

bash
git add .
git commit -m 'description'
git push origin xxx

3. 查看远程分支列表 ​

bash
git branch -a

4. 合并 xxx 分支到当前分支 ​

bash
git merge xxx

5. 删除分支 ​

bash
git branch -d xxx

6. 关联远程仓库分支 ​

bash
git push --set-upstream origin xxx

7. 从某个 commit 检出新分支 ​

bash
# 查看提交记录，输入 q 可以退出 git log
git log
git checkout commitId -b xxx

8. 解决分支冲突 ​

9. 生成 ssh-key ​

ssh 通常保存在 C:\Users\Administrator\.ssh 目录下，两个文件：

  • id_rsa 私钥
  • id_rsa.pub 公钥
bash
# 打开 git bash，进入.ssh 目录
cd ~/.ssh
# 生成 key，后面步骤可以 enter 跳过
ssh-keygen -t rsa -C "邮箱"

在 github settings → SSH and GPG keys 内 New SSH key 添加 key，内容就是 id_rsa.pub 文件内容

10. 配置用户名以及邮箱 ​

  • 全局，~/.gitconfig 中可以查看
bash
git config --global user.email <邮箱地址>
git config --global user.name <用户名>
  • 本地，项目目录下执行
bash
git config --local user.email <邮箱地址>
git config --local user.name <用户名>
