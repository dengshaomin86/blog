FFmpeg 视频旋转处理命令 ​

  • -vf 指令是一些简单的视频过滤器

顺时针旋转 90° ​

bash
ffmpeg -i input.mp4 -vf "transpose=1" input-r90.mp4

逆时针旋转 90° ​

bash
ffmpeg -i input.mp4 -vf "transpose=2" input-r270.mp4

水平翻转 ​

bash
ffmpeg -i input.mp4 -vf hflip input-hflip.mp4

垂直翻转 ​

bash
ffmpeg -i input.mp4 -vf vflip input-vflip.mp4
